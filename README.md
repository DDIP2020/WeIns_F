# front

This is front-end of project [WeIns](https://github.com/DDIP2020)

### Files structure:

```text
components/
    blog        // 动态展示框
    footer      // 底边栏
    issue       // 动态展示页面
    sidenav     // 侧边导航
    sign*Form   // 登录-注册的表单
    topnav      // 顶部导航栏
    backtop     // 回到顶部
    release     // 发布动态

views/
    catalog     // dynamic展示
    home        // 主页
    signup      // 注册界面
```
